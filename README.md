# 🏔️ 山海万相 · BestiAperture

**红外相机陷阱照片浏览与物种信息录入工具**

BestiAperture（山海万相）是一套面向野生动物监测项目的桌面工具，用于浏览红外相机陷阱拍摄的照片/视频，并快速完成物种信息标注与数据导出。

## ✨ 核心功能

- **文件夹浏览** — 按独立探测组管理子文件夹，自动加载照片与视频
- **高清预览** — 支持缩放、拖拽平移、全屏查看，右下角 3×3 九宫格快速定位
- **物种录入** — 起始编号 / 结束编号 / 物种 / 数量 / 性别，五字段快速录入
- **常用物种** — 最多 8 个快捷槽位（F1–F8），每个带独立颜色标识
- **事件标注** — 支持时间点事件和时间段事件，可独立导出为 txt
- **CSV 导出** — 一键导出物种汇总表，支持多文件夹合并
- **批处理工具集** — 内置照片整理、空拍剔除、多表合并、时间校正等子工具
- **9 套主题** — 灵感源自《山海经》神话（8 亮色 + 1 暗色），即时切换

## 🔄 完整工作流

红外相机监测数据的处理是一条从野外回收到最终成表的完整流水线。山海万相的所有功能都围绕这条主线设计：

```
回收 SD 卡 → 时间校正 → 照片分组 → 浏览录入 → 导出汇总 → 剔除空拍 → 数据合并
```

| 阶段 | 操作 | 使用工具 |
|------|------|----------|
| **1. 数据回收** | 从野外相机回收 SD 卡，按相机位点建立文件夹存放照片，文件命名中包含时间信息 | — |
| **2. 时间校正**（可选） | 若相机时钟有偏差，计算偏移量并修正文件名中的时间戳 | 时间校正 (TimeChaos) |
| **3. 照片整理** | 按时间间隔将散落照片自动分组到编号子文件夹，每个子文件夹 = 一次独立探测事件 | 照片整理 |
| **4. 浏览与录入** | 在主界面逐组浏览照片，标注物种名称、数量、性别 | **主界面**（核心环节） |
| **5. 导出汇总** | 导出物种汇总 CSV，作为后续处理的基础数据源 | 导出 CSV |
| **6. 剔除空拍**（可选） | 根据汇总中标记为「空」的记录批量删除空拍照片，释放存储空间 | 剔除空拍 |
| **7. 数据合并** | Bio-photo 提取 EXIF → TableWeaver 三表合并 → 生成最终影像数据汇总表 | Bio-photo + 数据处理 |

### 主界面操作步骤（对应阶段 4–5）

1. **选择父目录** — 点击工具栏「选择」或按 `Ctrl+O`
2. **浏览子文件夹** — 左侧列表点击加载照片
3. **查看照片** — 右侧缩略图网格或 `← / →` 方向键逐张浏览
4. **录入物种** — 底部填写信息，`Tab` 切换字段，`F1–F8` 快速填入常用物种
5. **保存记录** — 按 `Enter` 或点击「添加」
6. **导出汇总** — 点击「导出CSV」或按 `Ctrl+E`

## 🔧 内置工具

| 工具 | 说明 |
|------|------|
| 照片整理 | 按时间间隔自动分组到编号子文件夹，生成索引 CSV |
| 剔除空拍 | 根据汇总 CSV 中的「空」标记批量删除空拍照片 |
| 数据处理 (TableWeaver) | 物种汇总表 + 索引表 + Bio-Photo 原始数据 → 合并为最终影像汇总 |
| 时间校正 (TimeChaos) | 浏览器端计算器，校正相机时钟偏差（支持单点 / 两点线性插值） |

### 照片整理 — 支持的文件名格式

照片整理工具会在文件名的**任意位置**自动搜索日期时间信息，无需特定前缀，支持以下格式：

| 格式 | 示例 |
|------|------|
| `任意前缀 YYYY-MM-DD HH-MM-SS` | `猫 2024-01-01 12-00-00.jpg` |
| `任意前缀_YYYY-MM-DD_HH-MM-SS` | `旅行_2024-03-15_14-30-22.jpg` |
| `任意前缀_YYYYMMDD_HHMMSS` | `IMG_20240315_143022.jpg`、`PXL_20240315_143022.jpg` |
| `YYYYMMDDHHMMSS` | `20240315143022.jpg` |
| `YYYY-MM-DD`（仅日期） | `2024-03-15 猫咪.jpg` |

若文件名中无法识别日期时间，将依次尝试 EXIF 信息和文件修改时间作为回退方案。

### 关于 Bio-Photo

数据处理流程中的 EXIF 信息提取步骤依赖 **Bio-Photo (V2.1)**，该程序由北京大学李晟老师开发并免费提供。Bio-Photo **不包含在本软件中**，需用户自行获取并放置到程序同级目录下。

如需了解 Bio-Photo 的使用方法，请参阅李晟老师提供的《Bio-Photo (V2.1) 程序操作规程》。

## ❓ 常见问题

**Q: 视频文件怎么播放？**
选中视频文件后按 `Enter`，会用系统默认播放器打开。

**Q: 怎么批量处理空拍？**
先用 `Space` 标注默认物种为「空」→ 导出 CSV → 使用「剔除空拍」工具批量删除。

**Q: 录错了怎么办？**
在已录入列表中选中记录 → 点击「删除」→ 重新录入。

**Q: 照片整理支持哪些文件名？**
支持 `IMG_YYYYMMDD_HHMMSS`、`YYYY-MM-DD_HH-MM-SS`、任意前缀 + 日期时间等多种格式。文件名中包含日期时间信息即可自动识别，无需特定前缀。
